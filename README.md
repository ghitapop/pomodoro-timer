# Pomodoro Timer

A minimal, beautiful pomodoro timer web app.

## Features
- Timer Display with visual progress ring
- Timer Controls (Start/Pause/Reset/Skip)
- Session Types (Work 25min, Short Break 5min, Long Break 15min)
- Session Counter with localStorage persistence
- Audio Notification on timer completion

## Tech Stack
- Vanilla HTML/CSS/JavaScript
- No external dependencies

## Getting Started
```bash
./init.sh
```
Then open http://localhost:3000

## Design
- Dark theme with tomato red accent (#ff6347)
- Mobile-responsive centered layout
